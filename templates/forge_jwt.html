Use Forged Token

Copy the forged token above and replace the authentication cookie:

  1. Open browser DevTools (F12) → Storage tab (Firefox) or Application tab (Chrome)
  2. Click on Cookies → http://localhost:5000
  3. Find the cookie named token (NOT LocalStorage)
  4. Double-click the Value column and replace with your forged token
  5. Press Enter to save
  6. Refresh the page (F5) - you're now authenticated as the victim user!

Note: The server reads the JWT from the token cookie, not from LocalStorage. You must replace the cookie value to impersonate another user.
